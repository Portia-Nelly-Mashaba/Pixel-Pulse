What We Offer

Everything you need — gaming, connectivity, food, and learning — in one epic space.

Gaming Studio

Top-of-the-line consoles and custom gaming PC setups in a premium, air-conditioned environment. Hourly and daily packages available.

Internet Cafe

Blazing-fast fibre internet access. Print, scan, type documents, do job applications, and browse — all at affordable rates.

Vending Cafe

Grab snacks, hot drinks, cold beverages, and light meals. Stay fuelled while you game, study, or code.

Coding Classes

Fun and interactive coding lessons for kids aged 7–17. We teach HTML, Scratch, Python basics, and more in plain simple English.

Robotics Classes

Hands-on robotics workshops using LEGO kits and Arduino boards. Kids build real robots and learn engineering thinking.
